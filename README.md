TWITTER SENTIMENTAL ANALYSIS:
•	Analysis of tweets and engagement to segregate them into positive, negative and neutral.
•	Data ingestion was made by fetching historical tweets and real time tweets using the Twitter API.
•	The twitter data was processed through NLP to derive analytics on the sentiment of the tweets.

Extracted twitter data using tweepy - a Python library for accessing Twitter API Used Pandas to organize and handle this data. Performed basic statistics and created visualizations with numpy and matplotlib. Ran sentiment analysis of extracted (Trump's) tweets using textblob - a Python Library for processing textual data. This library provides a simple API for common natural language processing (NLP) tasks.
